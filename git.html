Tableau des commandes Git
Commande Description
git init Initialise le dépôt (attention : mettre le bon dossier)
git add . Ajoute toutes les modifications
git commit -m "explication" créer un nouveau commit. git add pousse les fichiers en zone d'index, git commit les sauvegarde réellement dans un nouveau commit
git log liste des commits
git log -n2 affiche les 2 derniers commits
git show sha-1 voir commit spécifique (cliquer sur la molette de la souris pour coller)
git checkout sha-1 remettre la version du sha-1
git checkout main remettre la version la plus récente
git clone "lien-github.com" récupérer un travail depuis un dépôt distant
git push -u origin main pousse les modifications vers le serveur
git push -f origin main pousse de force des modifications (déconseillé par beaucoup de dev
pwd savoir dans quel dossier je me trouve
mkdir "dossier" créer un dossier
touch "fichier.txt" créer un fichier
ls liste le dossier courant
ls -la liste tout plus précisément que ls
cd "dossier" aller dans le dossier
cd .. remonter d'un dossier dans l'arborescence
git status état du fichier
git diff affiche les modifications avant commit
git config --global user.name configurer son nom
git config --global user.email mon e-mail
git config --global --list affiche nom et mail
